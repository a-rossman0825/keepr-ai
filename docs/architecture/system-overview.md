## Project Overview:

### Summary
- A pinterest-like web application that allows users to create, share, and see "keeps" (photos) and "vaults" (albums).
- Users can create vaults to save their own or other user's "keeps" to.
- Users can archive their own vaults, making them private (only visible to the user).
- Application uses Auth0 for authentication workflow.

### Tech Stack

#### Front End:
- Vue 3
- TypeScript (Strict)
- Pinia Stores
- TailWindCSS
- SASS CSS
- Auth0 JWT Bearer Token Authentication

#### Back End:
- C#/.NET 
- Entity Framework Migrations
- MySQL Database
- Auth0 JWT Bearer Token Authentication
